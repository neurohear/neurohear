## 关于项目

NeuroHear 是一个基于深度学习的助听器算法工具包，旨在填补 OpenMHA/Tympan 在深度学习支持方面的空白。

### 核心特性

- **低延迟设计**: 目标延迟 <10ms，适合实时助听器应用
- **PyTorch 原生**: 所有算法基于 PyTorch 实现
- **ONNX 导出**: 支持导出为 ONNX，跨平台部署
- **模块化架构**: 降噪、反馈抑制、听力补偿等模块可灵活组合
